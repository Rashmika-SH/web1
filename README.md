## Setup Instructions

1. **Local Development**
   - Save the provided Lord Shiva Pujas image as `hero-image.jpg` in the same folder
   - Open `index.html` in a modern web browser
   - No build process required - pure HTML, CSS, and JavaScript

2. **Image Setup** (IMPORTANT)
   - Save your hero image as `hero-image.jpg` in the root folder
   - Recommended: 1920x1080px or higher resolution
   - Optimize to under 500KB for fast loading
   - See `SETUP-IMAGE.md` for detailed instructions

3. **Customization**
   - Replace placeholder phone number `+91XXXXXXXXXX` with actual number
   - Update email address `contact@astrodurgaprasad.com`
   - Add actual Google Maps embed code in the contact section
   - Update social media links

4. **Form Integration**
   - Uncomment the API call code in `script.js`
   - Set up backend endpoint for form submissions
   - Configure email notifications

5. **Deployment**
   - Upload all files including `hero-image.jpg` to your web hosting
   - Ensure HTTPS is enabled for security
   - Configure domain DNS settings
   - Set up SSL certificate
